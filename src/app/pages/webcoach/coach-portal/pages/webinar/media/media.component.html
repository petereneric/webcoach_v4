Cover-Bild

Dein Cover-Bild für diesen Online-Kurs wird u.a. in der Detail-Seite auf der Website von Webcoach angezeigt.

Am besten verwendest du ein Bild mit einer Auflösung von mindestens 400 x 200 Pixel und einer Dateigröße von höchstens 50 KB. Die Datei muss das Format WEBP haben. JPG und PNG Dateien kannst Du unter diesem Link in das WEBP Format umwandeln: https://convertio.co/de/jpg-webp/.

Thumbnail-Bild

Dein Thumbnail-Bild für diesen Online-Kurs wird u.a. in der Listen-Ansicht auf der Website von Webcoach angezeigt.

Am besten verwendest du ein Bild mit einer Auflösung von mindestens 100 x 50 Pixel und einer Dateigröße von höchstens 10 KB. Die Datei muss das Format WEBP haben. JPG und PNG Dateien kannst Du unter diesem Link in das WEBP Format umwandeln: https://convertio.co/de/jpg-webp/.

Trailer

Dein Video-Trailer für diesen Online-Kurs kann auf der Detail-Seite zu deinem Online-Kurs aufgerufen und abgespielt werden.

Am besten verwendest Du ein kurzes Video, das nicht länger als 30 Sekunden ist. Die Datei muss das Format mp4 haben.
